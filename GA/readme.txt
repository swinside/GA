2003년도 1학기 GA HW1
=====================

컴퓨터공학부 2003-21698 정우성



<Release>
GA Test 나 Graph 작업을 하기 위해서는 Release 폴더공간에서 해야합니다.
각각의 Test 나 Graph 또는 Check 작업을 위해서는 Release 폴더 안의 *.cmd 화일을 실행시키시면 됩니다.

<draw_appl> 에는 Visual Basic 으로 작성된 GA Graph drawing tool 이 있습니다.
Visual Studio 가 설치되어 있지 않다면, Install 에서 setup.exe 를 해야 정상적으로 동작합니다.
graph를 그리기 위해서는 graph실행을 통해 얻어진 xls 화일이 필요합니다.

<Document> 폴더에는 보다 자세한 내용을 담고 있는 문서가 들어 있습니다.

<Results> 에는 실험을 통해 생성된 결과화일이 있습니다.

<Graph> 폴더에는 각 종류별로 하나 또는 그 이상 ( zoom in ) 의 그래프 캡쳐화일이 들어 있습니다.
 graph 폴더안의 data 폴더는 graph 와 관련한 result 화일 및 xls 화일이 들어가 있습니다.

<screen_capture> 는 프로그램 실행과 관련한 화면 캡쳐 이미지들이 들어 있습니다.
